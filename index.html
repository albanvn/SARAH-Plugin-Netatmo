Le plugin Netatmo2 permet d'interroger les sondes NETATMO à l'aide de SARAH.

Pour cela, veuillez renseigner les paramètres du plugin Netatmo2 (compte NETATMO:email,mot de passe, id, secret).
Une fois la connexion effectuée avec le site de données de NETATMO, le fichier contenant les commande vocale reconnues netatmo2.xml sera mis à jour avec les paramètres de votre(vos) station(s) NETATMO (c'est à dire personnalisé avec le nom de chacune de vos balise NETATMO tels que configurés sur le site de NETATMO)
Vous pourrez alors interroger SARAH sur les informations relevées par vos balise NETATMO

Les autres paramètres du plugin NETATMO à configurer sont:
  • advice: permet d'activer (=1) ou de désactiver (=0) les conseils de SARAH après interrogation des balises lors du bilan de l'habitat ou remonté vocale de valeurs.
  • min_temp/max_temp: seuil de température min/max qui déclenche un conseil de SARAH
  • min_humidity/max_humidity: seuil de taux d'humidité min/max qui déclenche un conseil de SARAH
  • max_noise: seuil de bruit maximum qui déclenche un conseil de SARAH
  • max_co2: seuil de co2 maximum qui déclenche une alerte (disponible avec le plugin CRON pour Netatmo, SARAH-Plugin-NetatmoCron)
  • ignore_external: permet d'ignorer les paramètres température/humidité/bruit remonté par la balise exterieur NETATMO lors des conseils


Voici un exemple non exaustif des commandes vocales reconnues par le plugin Netatmo dans SARAH:

  • Combien fait-il ?
  • Vérifie la qualité de l'air/le taux d'humidité/la température
  • Synthèse NETATMO
  • quelles sont les données climatiques de la maison ?
  • Fait un bilan de l'habitat
  • Quel est le bilan de l'habitat ?
  • Quelle est la qualité de l'habitat ?
  • Bilan de l'habitat
  • Quelle est la {température/l'humidité/la qualité de l'air/la pression/le bruit} {dans/sur/de} {nom_balise/la maison} ?
  • Quelle est l'état des batteries Netatmo
